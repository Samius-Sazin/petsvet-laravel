File Structure explain
* app.php : responsible to create the application
* factory files : one factory for each model, define structure   of each field whenever we seed data, how seed data will generate.
* public->index.php is the entry state
* public folders are accessible to browser
* views files : all the html associated to roots
* storage, app : files generated by our code
* storage, framework : files generated by framework


** Necessary commands
* artisan commands
    php artisan list
* add new config
    php artisan config:publish

    php artisan config:publish configName

* create controller, in this case its WelcomeController
    php artisan make:controller WelcomeController

* Create model and migration, in this case its note
    php artisan make:model Note -m
